# Tests folder

## Purpose
This folder contains BDD (Behavior-Driven Development) test suites using pytest-bdd for validating export functionality and schema compliance.

## Structure
- **features/** – Gherkin feature files defining test scenarios
- **step_defs/** – Python step definitions implementing the test logic

## Files
- **features/brave_leo_export.feature** – BDD scenarios for Brave Leo export validation
- **step_defs/test_brave_leo_export.py** – Step implementations

## Running tests

pytest tests/
